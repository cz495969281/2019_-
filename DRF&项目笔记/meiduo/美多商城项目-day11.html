2. 项目部署

1）静态文件服务器

开发阶段: live-server

部署阶段: nginx

域名: www.meiduo.site

收集Django项目使用静态文件:

1. 配置收集静态文件保存目录STATIC_ROOT
2. python manage.py collectstatic

配置nginx提供静态文件​

2）后端API服务器

开发阶段：Django提供的开发web服务器 python manage.py runserver

部署阶段：uwsgi(遵循wsgi协议的web服务器)。

域名: api.meiduo.site

使用uwsgi:

1. 安装
    pip install uwsgi
2. 配置
3. 使用
    启动 uwsgi --ini uwsgi.ini
    停止 uwsgi --stop uwsgi.pid
