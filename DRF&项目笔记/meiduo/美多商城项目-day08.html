1. 购物车数据存储

需求:

​ 用户不管是否登录，都可以进行购物车记录添加。

登录用户购物车记录存储:

1）在redis中保存用户的购物车记录。

2）保存购物车记录时存什么数据？

​ 答: 商品id:数量 和 购物车记录勾选状态

3）采用redis中哪种数据格式？

答:

    # hash: 保存用户购物车记录中添加的商品id和对应数量
    cart_<user_id>: {
        <sku_id>: <value>,
        <sku_id>: <value>,
        ...
    }

       # set: 保存用户购物车记录勾选状态(保存勾选商品id)
    # 集合中元素唯一
    cart_selected_<user_id>: (<sku_id>, <sku_id>, ...)

    # 例如:
    cart_2: {
        '1': '3',
        '5': '2',
        '3': '1'
    }

    id为2的用户购物车记录:
        id为1的商品加了3件，
        id为5的商品加了2件，
        id为3的商品加了1件，

       cart_selected_2: ('1', '3')
    id为2的用户的购物车记录勾选状态:
        id为1商品和id为3的商品被勾选。

未登录用户的购物车记录存储:

redis数据存储服务器。

未登录用户的购物车数据服务器不进行保存，直接保存在客户端浏览器。

COOKIE: 由服务器生成，保存在浏览器。

{
    '<sku_id>': {
        'count': '<count>',
        'selected': '<selected>',
    },
    '<sku_id>': {
        'count': '<count>',
        'selected': '<selected>',
    },
    ...
}

json:

​ json.dumps(dict): 将python字典转换为json字符串。

​ json.loads(json_str): 将json字符串转换为python字典。

pickle:

​ pickle.dumps(obj): 将obj转换成bytes字节流。

​ pickle.loads(bytes字节流)：将bytes字节流转换为obj。

base64:

​ base64.b64encode(bytes字节流)：将bytes字节流进行base64编码，返回编码之后内容(bytes字节流)

​ base64.b64decode(bytes字节流&str): 将参数内容进行base64解码，返回解码之后内容(bytes字节流)

设置购物车cookie数据:

​ res = base64.b64encode(pickle.dumps(cart_dict)).decode()

解析cookie中购物车数据:

​ res = pickle.loads(base64.b64decode(cart_data))

2. 购物车记录添加
API: POST /cart/
参数:
    通过请求头传递jwt token数据给服务器
    {
        'sku_id': '商品id',
        'count': '数量',
        'selected': '勾选状态'
    }
响应:
    {
        'sku_id': '商品id',
        'count': '数量',
        'selected': '勾选状态'
    }

API: PUT /cart/selection/

7. redis命令

hash命令:

1）hincrby(key, field, value)

​ 给hash中field属性的值增加value，如果field属性不存在，创建新的属性和值

2）hgetall(key)

​ 获取hash中所有的属性和值

3）hset(key, field, value)

​ 将hash中指定field属性值设置为value

4）hdel(key, *fields)

​ 删除hash中指定的field属性和值，如果属性不存在，直接忽略

5）hkeys(key)

​ 获取hash中所有属性

set命令:

1）sadd(key, *members)

​ 向set集合中添加元素，不需要考虑是否重复，set中元素唯一

2）smembers(key)

​ 获取set集合中的所有元素

3）srem(key, *members)

​ 从set集合中移除元素，有则移除，无则忽略
